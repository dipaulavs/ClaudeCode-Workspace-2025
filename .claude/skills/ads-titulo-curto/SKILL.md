# Ads Titulo Curto - Meta Ads Copy Generator

## Overview

Generate professional Meta Ads copy for Facebook/Instagram campaigns using proven Alex Hormozi frameworks from $100M Offers and $100M Leads. Creates complete ad text structured for Meta's ad fields: Main Text (hook), Short Title, Description (body), and CTA.

## When to Use This Skill

Auto-invoke when the user requests:
- Meta Ads copy creation
- Facebook/Instagram ad text
- Ad copy for any niche (real estate, courses, services, products, etc.)
- Carousel, static image, or reels ad formats
- Mentions needing ad copy or creative text

## Workflow

### Step 1: Gather Input

Ask the user for required information:

**REQUIRED:**
1. **Niche/Product/Offer** - What are you selling? What's the specific offer?
2. **Format** - Static image, carousel (multiple slides), or reels (video)?
3. **CTA Style** - How should the audience respond?
   - **Keyword in comments** (e.g., "Comment 'CASA' to receive details")
   - **WhatsApp button click** (direct message via WhatsApp CTA)
   - **DM/Inbox** (send message via Messenger/Instagram DM)

**OPTIONAL (if user provides):**
- Target avatar (who is this for?)
- Main pain point or objection to address
- Specific pricing or offer details
- Urgency/scarcity elements

### Step 2: Select Hormozi Framework

Based on input, choose the most appropriate framework from the examples in `references/`:

**For hooks (Main Text):**
- Specific + Time + Proof (recent results with numbers)
- Math That Makes Sense (simple comparison)
- Objection Destroyed (address #1 barrier immediately)
- Loss Framing (cost of not acting)
- Insane Comparison (relatable item vs offer)
- The "Secret" (insider info they don't want you to know)
- Before/After Brutal (transformation story)
- Do The Math With Me (interactive calculation)

**For titles:**
- Direct benefit statement
- Curiosity hook
- Question format
- Urgency/scarcity signal

**For body copy (Description):**
- Pain â†’ Agitate â†’ Solve
- Comparison (current state vs future state)
- Math breakdown (show ROI clearly)
- Qualification (filter hard, qualify audience)
- Social proof (specific cases/names/numbers)

3. **Description**: 300-600 characters (appears in expanded view)
