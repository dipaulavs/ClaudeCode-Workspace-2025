# Creative Angles - Meta Ads (Validados)

## Ângulos de Criativos de Alta Performance

Este documento contém 15+ ângulos validados de criativos para Meta Ads, extraídos de campanhas reais de alto desempenho.

---

## 1. Dor/Problema (Pain Point)

**Objetivo:** Gerar identificação imediata com o problema do avatar.

**Estrutura:**
- Hook: Expor a dor específica
- Body: Agitar a dor (mostrar consequências)
- CTA: Oferecer solução

**Exemplos:**

**Relacionamento:**
```
Hook: "Ele se afastou de você e você não entende por quê?"
Body: "Você envia mensagem, ele demora horas pra responder...
       Você tenta conversar, ele está sempre 'ocupado'...
       Você se sente ignorada, invisível, insegura."
CTA: "Descubra o que fazer agora → [link]"
```

**Fitness:**
```
Hook: "Treina todo dia mas a barriga não sai?"
Body: "Você malha 5x por semana, come salada, faz cardio...
       Mas o peso não desce. A barriga continua lá.
       Frustrante, né?"
CTA: "Veja o erro que 97% comete → [link]"
```

**Nicho:** Relacionamento, Saúde, Fitness, Negócios

## 2. Desejo/Transformação (Desire/Dream Outcome)

**Objetivo:** Despertar esperança e visualização do resultado ideal.

**Estrutura:**
- Hook: Mostrar resultado desejado
- Body: Pintar cenário do sonho realizado
- CTA: Mostrar que é possível

**Exemplos:**

**Negócios:**

```
       Trabalhar de pijama, no seu tempo...
       Ver o dinheiro caindo na conta todo dia."
CTA: "Descubra como → [link]"
```

**Nicho:** Universal (todos os nichos)

## 3. Quebra de Crença/Educacional (Belief Break)

**Objetivo:** Desconstruir crença limitante que impede a compra.

**Estrutura:**
- Hook: Afirmar que não é culpa deles
- Body: Explicar o "verdadeiro" motivo
- CTA: Revelar a solução real

**Exemplos:**

**Relacionamento:**
```
Hook: "Não é culpa sua que ele se afastou. É porque..."
Body: "A sociedade te ensinou ERRADO sobre homens.
       Te disseram pra ser 'difícil', jogar joguinhos...
       Mas homens ODEIAM isso. Vou te mostrar o que funciona."
CTA: "Assista o vídeo completo → [link]"
```

**Emagrecimento:**
```
Hook: "Não é falta de força de vontade. É seu METABOLISMO."
Body: "Te ensinaram a comer menos e malhar mais...
       Mas isso DESTRÓI seu metabolismo.
       Vou te mostrar o caminho certo."
CTA: "Descubra a verdade → [link]"
```

**Nicho:** Saúde, Fitness, Relacionamento, Negócios

```

       30 dias depois: R$50.437 de faturamento."
CTA: "Assista o case completo → [link]"
```

**Nicho:** Universal (todos os nichos)

## Como Usar Este Documento

1. **Identificar Avatar:** Defina quem é seu cliente ideal
2. **Mapear Dor/Desejo:** Qual maior problema/desejo dele?
3. **Escolher 3-4 Ângulos:** Baseado na matriz acima
4. **Criar 4-6 Criativos por Ângulo:** Para 1 batch de teste
5. **Testar no Meta Ads:** Testing Campaign (CBO, 2-3 dias)
6. **Identificar Winners:** CPA ≤ target + ROAS ≥ target
7. **Escalar:** Mover winners para Scaling Campaign

**Regra de Ouro:** 50% big swings (ângulos novos) + 50% iterations (variações de winners)
