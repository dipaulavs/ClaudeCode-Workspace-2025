# Hormozi Frameworks - Aplicação em Meta Ads

Frameworks extraídos dos livros $100M Offers, $100M Leads e $100M Money Models, aplicados especificamente para campanhas Meta Ads.

## 1. Value Equation (Equação de Valor)

**Fonte:** $100M Offers

### Fórmula:

```
Valor Percebido = (Dream Outcome × Perceived Likelihood) / (Time Delay × Effort & Sacrifice)
```

### Componentes:

#### 1.1 Dream Outcome (Resultado dos Sonhos)

O que o cliente REALMENTE quer.

**Aplicação em Meta Ads:**
- Hook do criativo = Dream Outcome
- "Imagine ele te ligando dizendo que sente sua falta" (Relacionamento)
- "R$10.000/mês trabalhando de casa" (Negócios)
- "85kg → 65kg em 12 semanas" (Fitness)

**Como descobrir:**
- Perguntar: "Se pudesse resolver este problema com uma varinha mágica, qual seria o resultado?"
- Ir além do óbvio: "emagrecer" → "sentir-se confiante na praia"

#### 1.2 Perceived Likelihood (Probabilidade Percebida)

Quão provável o cliente acredita que vai conseguir o resultado.

**Aplicação em Meta Ads:**
- Prova social no criativo (depoimentos, números)
- "Mais de 10.000 mulheres já reconquistaram usando este método"
- Garantia forte ("dinheiro de volta em dobro")
- Case studies no Body do vídeo

**Como aumentar:**
- Mostrar resultados de pessoas similares
- Reduzir objeções antecipadamente
- Demonstrar método comprovado

#### 1.3 Time Delay (Atraso de Tempo)

Quanto tempo até obter o resultado.

**Aplicação em Meta Ads:**
- Especificar prazo no hook
- "Como Maria fez João voltar em 7 dias"
- "Primeira venda em 48 horas"
- "Resultados visíveis em 2 semanas"

**Como reduzir:**
- Prometer resultado mais rápido (se honesto)
- Entregar valor imediato (lead magnet)
- Mostrar progresso incremental

#### 1.4 Effort & Sacrifice (Esforço e Sacrifício)

O que o cliente precisa fazer/desistir para obter resultado.

**Aplicação em Meta Ads:**
- Reduzir objeções de esforço no copy
- "Sem dieta restritiva" (Fitness)
- "7 minutos por dia" (Treino)
- "Sem precisar aparecer em vídeo" (Negócios digitais)

**Como reduzir:**
- Simplificar o processo
- Automatizar partes do método
- Remover barreiras de entrada

### Exemplo Completo - Value Equation em Criativo:

**Nicho:** Reconquista (Relacionamento)

```
Dream Outcome: "Ele volta implorando por você"
Perceived Likelihood: "10.000+ mulheres já reconquistaram"
Time Delay: "Primeiros sinais em 3-7 dias"
Effort & Sacrifice: "Sem joguinhos ou fingir desinteresse"

CRIATIVO:
Hook: "Como fazer ele voltar em 7 dias (10.000+ mulheres já fizeram)"
Body: "Você NÃO precisa jogar joguinhos ou fingir desinteresse.
       Método simples. 3 passos. Primeiros sinais em 3-7 dias.
       Maria fez, João voltou implorando."
CTA: "Descubra o método → [link]"
```

## 2. Grand Slam Offer (Oferta Irresistível)

**Fonte:** $100M Offers

### Componentes de uma Grand Slam Offer:

#### 2.1 The Stack (Empilhamento de Valor)

Somar valor de cada componente individualmente.

**Aplicação em Landing Page (após clique do Meta Ad):**

```
VOCÊ VAI RECEBER:

✅ Módulo 1: Método Reconquista Consciente (R$997)
✅ Módulo 2: Comunicação Magnética (R$497)
✅ Módulo 3: Criando Atração Reversa (R$497)

BÔNUS:
✅ Bônus 1: Script Pronto de Mensagens (R$297)
✅ Bônus 2: Checklist Reconquista 7 Dias (R$197)
✅ Bônus 3: Comunidade Exclusiva VIP (R$497/ano)

VALOR TOTAL: R$2.982
VOCÊ PAGA HOJE: R$197

+ Garantia 30 dias (dinheiro de volta)
```

**Aplicação no Meta Ad:**
- Teaser do stack no Body do vídeo
- "Método completo + 3 bônus exclusivos"

#### 2.2 Scarcity & Urgency (Escassez e Urgência)

Criar FOMO genuíno (não fake).

**Tipos de Escassez:**

**Tempo:**
- "Oferta expira em 24h"
- "Carrinho fecha às 23h59 de hoje"
- Countdown timer na landing page

**Quantidade:**
- "Apenas 50 vagas disponíveis"
- "Restam 7 vagas" (atualizar dinamicamente)

**Bônus:**
- "Bônus X apenas para os 20 primeiros"
- "Fast Action Bonus (primeiras 24h)"

**Aplicação no Meta Ad:**
```
Hook: "Últimas 10 vagas. Fecha hoje às 23h59."
Body: "50 vagas abertas. 40 já preenchidas.
       Próxima turma só em 6 meses.
       Se quer entrar, tem que ser AGORA."
CTA: "Garanta sua vaga → [link]"
```

#### 2.3 Guarantee (Garantia)

Reverter risco para você.

**Tipos de Garantia:**

**Money Back:**
- "30 dias de garantia incondicional"
- "Não funcionou? Devolvemos em DOBRO"

**Resultado:**
- "Garantia de resultado em 90 dias ou seu dinheiro de volta"

**Try-Before-You-Buy:**
- "7 dias grátis (cancele quando quiser)"
- "Primeira aula grátis"

**Aplicação no Meta Ad:**
```
CTA: "Teste GRÁTIS por 7 dias (sem cartão) → [link]"
```

#### 2.4 Naming (Nomenclatura)

Nome memorável e diferenciado.

**Fórmula:**
- [Resultado] + [Método/Sistema]
- [Avatar] + [Transformação]

**Exemplos:**
- "Método Reconquista Consciente" (Relacionamento)
- "Sistema 7 Minutos de Queima Metabólica" (Fitness)
- "Funil Evergreen Escalável" (Negócios)

**Aplicação no Meta Ad:**
```
Hook: "Descubra o Método Reconquista Consciente"
(Não: "Descubra como reconquistar")
```

**Nicho:** Infoproduto de Negócios Digitais

```

O QUE VOCÊ VAI RECEBER:
✅ Módulo 1: Oferta Irresistível (R$1.997)
```

## 3. Core Four (4 Formas de Gerar Leads)

**Fonte:** $100M Leads

### Visão Geral:

```
┌─────────────────────┐
│   CORE FOUR         │
├─────────────────────┤
│ 1. Warm Outreach    │ → Contatos quentes (já te conhecem)
│ 2. Cold Outreach    │ → Contatos frios (não te conhecem)
│ 3. Paid Ads         │ → Anúncios pagos (Meta, Google, etc)
│ 4. Content/Organic  │ → Conteúdo orgânico (SEO, social)
└─────────────────────┘
```

### 3.3 Paid Ads (Foco desta skill)

**Aplicação Meta Ads:**

Meta Ads é a execução principal do Core Four #3.

**Estratégia:**
1. **Testing Campaign** (descobrir o que funciona)
   - 20-40% do budget
   - Testar ângulos diferentes
   - Identificar winners

2. **Scaling Campaign** (escalar o que funciona)
   - 60-80% do budget
   - Apenas criativos validados
   - Otimização contínua

3. **Remarketing** (reengajar warm audience)
   - 10-20% do budget
   - Quem visitou mas não comprou
   - Oferta + urgência

**Integração com outros Core Four:**

```
Meta Ads → Landing Page → Lead Magnet → Email (Warm Outreach)
Meta Ads → Webinar → Offer → Follow-up (Warm Outreach)
Meta Ads → Conteúdo → Engajamento → Orgânico (Content)
```

## 4. Lead Getters (Amplificadores)

**Fonte:** $100M Leads

### Visão Geral:

```
┌────────────────────────────┐
│   LEAD GETTERS             │
├────────────────────────────┤
│ 1. Customers (Clientes)    │ → Indicação, reviews, testimonials
│ 2. Employees (Equipe)      │ → Time vendendo ativamente
│ 3. Agencies (Agências)     │ → Terceirizar aquisição
│ 4. Affiliates (Afiliados)  │ → Comissão por venda
└────────────────────────────┘
```

### Aplicação em Meta Ads:

#### 4.1 Customers (Clientes Atuais)

**Como usar:**
- UGC (User Generated Content) nos criativos
- Depoimentos em vídeo
- Case studies como creative angles
- Reviews sociais (comentários, DMs)

**Exemplo:**
```
Criativo = Depoimento de cliente em vídeo
Hook: "Comprei achando que não ia funcionar... (antes/depois)"
Body: Cliente mostra resultado + experiência
CTA: "Você também pode → [link]"
```

#### 4.2 Affiliates (Programa de Afiliados)

**Como usar:**
- Criar programa de afiliados
- Dar criativos prontos para afiliados
- Afiliados rodam próprios Meta Ads
- Você paga comissão por venda

**Estrutura:**
```
Você cria: Oferta + Funil + Criativos (base)
Afiliado: Roda Meta Ads com link rastreado
Resultado: Afiliado ganha 30-50% de comissão
Benefício: Você escala sem gastar seu próprio budget
```

## 5. Money Models (Modelos de Monetização)

**Fonte:** $100M Money Models

### 5.1 Attraction Offers (Ofertas de Atração)

Objetivo: Adquirir clientes (break-even ou lucro pequeno)

**Free + Shipping:**
```
Meta Ad: "Livro grátis (pague apenas R$19,90 de frete)"
Objetivo: Adquirir cliente pagante por R$20
Backend: Upsells na thank you page (+ R$80 AOV)
```

**Decoy Offer:**
```
Landing Page:
□ Plano Basic: R$97/mês
☑ Plano Pro: R$197/mês ← (parece barato comparado)
□ Plano Enterprise: R$997/mês
```

**Win Your Money Back:**
```
Meta Ad: "Compre agora. Não funcionar? Devolvemos em DOBRO"
Efeito: Reduz risco, aumenta conversão
```

### 5.2 Upsell Offers (Aumentar Ticket)

Objetivo: Aumentar AOV (Average Order Value) por cliente

**Classic Upsell:**
```
Thank You Page (após compra):
"Parabéns! Quer adicionar [complemento] por apenas +R$97?"

Exemplo:
Comprou: Curso de Ads (R$497)
Upsell: Templates de Copy (+R$97)
AOV: R$594 (↑ 19%)
```

**Anchor Upsell:**
```
Checkout:
1º - Mostrar R$997 (caro)
2º - "Desconto especial: R$497" (parece barato)
```

### 5.3 Continuity Offers (Receita Recorrente)

Objetivo: LTV alto (lifetime value)

**Membership/Subscription:**
```
Oferta Principal: R$997 (único)
Upsell: Comunidade VIP R$97/mês

Cliente médio fica 8 meses = R$776 adicional
LTV: R$997 + R$776 = R$1.773
```

**Continuity Bonus:**
```
"Fique 6 meses na comunidade = ganhe [bônus R$497]"
Efeito: Reduz churn, aumenta LTV
```

## 6. Aplicação Integrada (Framework Completo)

### Exemplo: Infoproduto R$497

**1. Value Equation:**
```
Dream Outcome: "Faturar R$10k/mês online"
Perceived Likelihood: "500+ alunos já faturaram R$10k+"
Time Delay: "Primeira venda em 30 dias"
Effort: "2h/dia de dedicação"
```

**2. Grand Slam Offer:**
```
Método [Nome]
Módulos 1-4 (R$4.000 de valor)
Bônus 1-3 (R$2.000 de valor)
Valor Total: R$6.000
Você paga: R$497
Garantia: 30 dias dinheiro de volta
Vagas: 100 (escassez)
Prazo: 72 horas (urgência)
```

**3. Meta Ads (Core Four #3):**
```
Testing Campaign:
Batch 1 - Ângulo Dor (4 criativos)
Batch 2 - Ângulo Desejo (4 criativos)
Batch 3 - Ângulo Prova Social (4 criativos)

Scaling Campaign:
Winners (até 50 criativos ativos)

Budget: R$500/dia
├── R$350/dia Scaling (70%)
├── R$100/dia Testing (20%)
└── R$50/dia Remarketing (10%)
```

**4. Money Models:**
```
Attraction: R$497 (produto principal)
↓
Upsell 1: +R$197 (templates) → 30% conversão → +R$59 AOV
↓
Upsell 2: +R$997 (mentoria) → 10% conversão → +R$100 AOV
↓
Continuity: R$97/mês (comunidade) → 40% conversão → 8 meses → +R$310 LTV

LTV Total: R$497 + R$59 + R$100 + R$310 = R$966
```

**5. Métricas:**
```
CPA Target: R$100 (10% do LTV)
ROAS Target: 5.0x (R$5 retornados para cada R$1 gasto)
Budget Máximo: R$966 LTV × 10% = R$96 por cliente
```

## 7. Checklist de Aplicação

Antes de lançar campanha Meta Ads, validar:

### Value Equation ✅
- [ ] Dream Outcome claramente definido?
- [ ] Perceived Likelihood reforçada (provas)?
- [ ] Time Delay especificado?
- [ ] Effort & Sacrifice minimizado?

### Grand Slam Offer ✅
- [ ] The Stack criado (valor empilhado)?
- [ ] Scarcity real (não fake)?
- [ ] Urgency real (não fake)?
- [ ] Garantia forte (reverter risco)?
- [ ] Naming memorável?

### Core Four (Paid Ads) ✅
- [ ] Testing Campaign estruturada?
- [ ] Scaling Campaign estruturada?
- [ ] Remarketing Campaign estruturada?
- [ ] Budget split definido?

### Money Models ✅
- [ ] Attraction Offer definida?
- [ ] Upsells mapeados?
- [ ] Continuity planejada?
- [ ] LTV calculado?
- [ ] CPA máximo definido?

### Creative Diversity ✅
- [ ] 3+ ângulos diferentes escolhidos?
- [ ] 4-6 criativos por ângulo?
- [ ] 50% big swings + 50% iterations?
- [ ] Hooks validados?
- [ ] CTAs claros?

---

**Nota:** Aplicar estes frameworks de forma integrada multiplica resultados exponencialmente. Não usar apenas 1 framework isolado.
