# Hormozi Meta Ads Strategist

## Overview

Estrategista completo de Meta Ads combinando frameworks validados de Alex Hormozi ($100M Offers, $100M Leads, $100M Money Models) com o sistema Andromeda (algoritmo 2025+). Gera estratégias de campanhas completas incluindo pixel, remarketing, criativos de valor, segmentação por ângulos e otimização baseada em creative diversity.

Combina 3 pilares fundamentais:

```
┌─────────────────┐    ┌──────────────────┐    ┌─────────────────┐
```

### 1. Hormozi Frameworks (Foundation)

**Value Equation** - Aumentar valor percebido:
- Dream Outcome (resultado desejado)
- Perceived Likelihood (probabilidade percebida)
- Time Delay (reduzir tempo)
- Effort & Sacrifice (reduzir esforço)

**Grand Slam Offer** - Oferta irresistível:
- Descobrir o que o avatar realmente quer
- Aumentar valor percebido vs preço pedido
- Criar urgência e escassez verdadeiras
- Stack de valor (bônus estratégicos)

**Core Four** (Lead Generation):
- Warm Outreach (contatos quentes)
- Cold Outreach (contatos frios)
- Paid Ads (anúncios pagos) ← FOCO
- Content (conteúdo orgânico)

**Lead Getters** (Amplificação):
- Customers (clientes atuais)
- Employees (colaboradores)
- Agencies (agências)
- Affiliates (afiliados)

### 2. Meta Andromeda System (Execution)

Sistema de Machine Learning de 2025+ focado em **Creative Diversity**:

**Princípio Central:** Algoritmo não busca mais "1 criativo campeão" → busca **múltiplos criativos** atingindo **diferentes segmentos** da audiência.

**Estrutura Obrigatória:**
```
Testing Campaign (20-40% budget)
├── Batch 1 (Ângulo: Dor/Problema)
├── Batch 2 (Ângulo: Desejo/Transformação)
├── Batch 3 (Ângulo: Quebra de Crença)
├── Batch 4 (Ângulo: Prova Social)
└── ...

Scaling Campaign (60-80% budget)
└── Ad Set Único (Broad)
    ├── Winner 1 (do Batch X)
    ├── Winner 2 (do Batch Y)
    ├── Winner 3 (do Batch Z)
    └── ... (até 50 criativos)
```

**Targeting:** SEMPRE BROAD + Creative Diversity para atingir diferentes segmentos.

**Creative Diversity Validada:**
- Cada criativo deve atingir um segmento/ângulo DIFERENTE
- Não fazer iterações do mesmo criativo (exceto 50% das vezes)
- Mínimo 4-6 criativos novos por semana (1 batch)
- 50% big swings (totalmente novo) + 50% iterations (variações do que funciona)

## Workflow: Criar Campanha Completa

Quando usuário solicitar estratégia Meta Ads, seguir este fluxo:

### Step 1: Coleta de Informações

Perguntar SEMPRE:

1. **O que você vende?** (produto/serviço/infoproduto)
2. **Preço/Ticket?** (valor do produto)
3. **Quanto quer investir?** (budget diário/total)
4. **Objetivo?** (leads, vendas, agendamentos, eventos offline)
5. **Avatar?** (quem é o cliente ideal? dor principal?)
6. **Já tem criativos?** (vídeos/imagens prontos?)
7. **Pixel configurado?** (Facebook Pixel + Conversions API)

### Step 2: Análise Hormozi (Value Equation + Grand Slam Offer)

Aplicar frameworks:

**Value Equation:**
```
Valor = (Dream Outcome × Perceived Likelihood) / (Time Delay × Effort & Sacrifice)
```

Identificar:
- Qual o resultado dos sonhos do avatar?
- Como aumentar probabilidade percebida? (provas, garantias)
- Como reduzir tempo até resultado?
- Como reduzir esforço/sacrifício?

**Grand Slam Offer:**
- Criar oferta irresistível baseada na Value Equation
- Stack de valor (bônus que amplificam resultado)
- Urgência real (não fake scarcity)
- Garantia poderosa (reverse risk)

### Step 3: Definir Ângulos de Criativos

Criar matriz de ângulos baseada em segmentos:

**Ângulos Validados:**

1. **Dor/Problema** - "Ele se afastou e você não entende por quê?"
2. **Desejo/Transformação** - "Imagine ele te ligando dizendo que sente sua falta"
3. **Quebra de Crença/Educacional** - "Não é culpa sua que ele se afastou. É porque..."
4. **Prova Social** - "Como Maria fez João voltar em 7 dias"
5. **Autoridade** - "Método validado por 10.000+ mulheres"
6. **Mecânica Única** - "O erro nº1 que faz homens se afastarem"
7. **Before/After** - "Antes: ele sumia. Depois: ele pede para casar"
8. **Benefício Específico** - "Faça ele te desejar 3x mais (sem parecer desesperada)"
9. **Curiosidade/Gap** - "A palavra que faz qualquer homem te procurar"

**Para cada ângulo, criar:**
- Hook (primeiros 3 segundos)
- Body (15-30 segundos mostrando transformação)
- CTA (call to action claro)

### Step 5: Critérios de Validação (Winners)

**Criativo se torna WINNER quando:**

```
Métricas Mínimas:
├── Conversões: 10+ (double digits)
├── Spend: 5x AOV mínimo
├── CPA: ≤ Target CPA
├── ROAS: ≥ Target ROAS (geralmente 2.5x+)
└── Volume: consistente por 2-3 dias
```

**Quando mover para Scaling:**
1. Criativo atinge métricas acima ✅
2. Copiar POST ID (manter engajamento social)
3. Duplicar AD ID no Scaling Campaign
4. Manter ativo no Testing também (não pausar)
5. Monitorar por 3-7 dias no Scaling

**Quando pausar criativo:**
- Spend > 3-4x AOV sem conversão
- CPA > 2x Target CPA
- ROAS < 1.0x por 3+ dias consecutivos
- Ad fatigue (performance cai 50%+ por 5+ dias)

### Step 6: Otimização Contínua

**Workflow Semanal:**

```
Segunda-feira: Lançar Batch novo (4-6 criativos, 1 ângulo)
├── 50% big swings (totalmente novo)
└── 50% iterations (variações de winners)

Terça-Quarta: Monitorar Testing Campaign
├── Identificar early winners (CPA baixo + volume)
└── Pausar clear losers (spend > 3x AOV, zero vendas)

Quinta-Sexta: Promover Winners → Scaling
├── Copiar POST ID (manter engajamento)
├── Duplicar no Scaling Campaign
└── Limpar Scaling (pausar criativos sem spend)

Sábado-Domingo: Análise & Planejamento
├── Revisar métricas gerais (ROAS, CPA, LTV)
├── Identificar ângulos que funcionam melhor
└── Planejar batches da próxima semana
```

**Métricas-Chave:**
- **CPA** (Cost Per Acquisition) - quanto custa 1 cliente
- **ROAS** (Return On Ad Spend) - quanto retorna para cada R$1 gasto
- **CTR** (Click-Through Rate) - % que clica no ad
- **Hook Rate** (primeiros 3s) - % que assiste 3+ segundos
- **Retention Rate** - % que assiste vídeo inteiro
- **CPM** (Cost Per Mille) - custo por 1000 impressões

## Money Models (Maximizar LTV)

Aplicar modelos do livro $100M Money Models para aumentar receita por cliente:

### Attraction Offers (Adquirir Clientes)

**Free + Shipping** - Cliente paga apenas envio
```
Exemplo: "Livro grátis (pague apenas R$19,90 de frete)"
Meta Ads: Budget agressivo, CPA break-even OK
Backend: Upsells na thank you page
```

**Win Your Money Back** - Garantia reversa
```
Exemplo: "Compre agora. Se não funcionar, devolvemos em dobro"
Meta Ads: Reduz objeção, aumenta conversão
```

**Decoy Offer** - Opção falsa que valoriza oferta principal
```
Exemplo:
- Plano Basic: R$97/mês ❌
- Plano Pro: R$197/mês ✅ (mais popular)
- Plano Enterprise: R$997/mês
Meta Ads: Anunciar Plano Pro (decoy = Basic)
```

### Upsell Offers (Aumentar Ticket Médio)

**Classic Upsell** - Após compra, oferecer upgrade
```
Thank You Page: "Adicione [produto complementar] por apenas +R$XX"
Aumenta AOV em 30-50%
```

**Anchor Upsell** - Mostrar opção cara primeiro
```
Checkout: Mostrar R$997 → depois R$497 (parece barato)
```

**Menu Upsell** - Múltiplas opções após compra
```
"Escolha seu upgrade:"
□ Bônus A (+R$97)
□ Bônus B (+R$147)
□ Bônus C (+R$197)
```

### Continuity Offers (Receita Recorrente)

**Membership/Subscription** - Mensalidade
```
Exemplo: R$97/mês para acessar comunidade + atualizações
LTV sobe drasticamente
```

**Continuity Bonus** - Incentivo para continuar
```
"Permaneça 6 meses = ganhe [bônus valioso]"
Reduz churn
```

## Auto-Correction System

This skill includes an automatic error correction system that learns from mistakes and prevents them from happening again.

### How It Works

When a script or command in this skill fails:

1. **Detect the error** - The system identifies what went wrong
2. **Fix automatically** - Updates the skill's code/instructions
3. **Log the learning** - Records the fix in LEARNINGS.md
4. **Prevent recurrence** - Same error won't happen again

### Using Auto-Correction

**Scripts available:**

```bash
# Fix a problem in this skill's SKILL.md
python3 scripts/update_skill.py <old_text> <new_text>

# Log what was learned
python3 scripts/log_learning.py <error_description> <fix_description> [line]
```

**Example workflow when error occurs:**

```bash
# 1. Fix the error in SKILL.md
python3 scripts/update_skill.py \
    "--prompt" \
    ""

# 2. Log the learning

    "SKILL.md:line_number"
```
