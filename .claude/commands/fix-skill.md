# Fix Skill

Ativa automaticamente a skill `fix-skill` para corrigir erros em skills.

Esta skill detecta e corrige erros automaticamente através de:
1. Detecção da skill que falhou
2. Análise do erro
3. Correção em SKILL.md
4. Registro em LEARNINGS.md
5. Prevenção de recorrência
